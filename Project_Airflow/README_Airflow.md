# 🚕 Построение витрины данных для Яндекс Такси

## 📌 Описание проекта

Автоматизированная обработка данных о поездках такси с агрегацией по способам оплаты.

**Задача:** Построить витрину данных для финансовой и продуктовой команд.

**Результат:** Ежедневно обновляемая таблица с показателями:
- Количество поездок
- Средняя стоимость
- Средние чаевые
- Суммарная выручка

## 🏗 Архитектура
S3 (данные) → Airflow (планировщик) → Spark (обработка) → ClickHouse (витрина)
